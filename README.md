# Quantum Computing Learning

Structured learning path for quantum computing fundamentals and algorithms.

## Objective

Building foundational knowledge in quantum computing principles, quantum algorithms, and their practical implementations using Qiskit.

## Repository Structure

### Fundamentals
**Location:** `01-fundamentals/`  
**Status:** In Progress  
**Description:** Core quantum computing concepts including quantum gates, superposition, entanglement, and measurement principles.  
**Topics:**
- Quantum gates (Hadamard, Pauli, CNOT)
- Superposition and measurement collapse
- Quantum entanglement and Bell states
- Bloch sphere visualization

### Algorithms
**Location:** `02-algorithms/`  
**Status:** Planned  
**Description:** Implementation of fundamental quantum algorithms.  
**Planned Topics:**
- Grover's search algorithm
- Deutsch-Jozsa algorithm
- Quantum Fourier Transform basics

## Setup
```bash
python3 -m venv quantum-env
source quantum-env/bin/activate
pip install -r requirements.txt
```

## Learning Resources

- IBM Qiskit Textbook
- Nielsen & Chuang - Quantum Computation and Quantum Information
- Griffiths - Introduction to Quantum Mechanics
